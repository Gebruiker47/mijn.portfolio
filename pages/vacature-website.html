Deze pagina bevat de informatie over mijn vacature applicatie. Ongeveer 3 jaar geleden heb ik deze applicatie als hobby gebouwd. De technische informatie zijn hieronder opgesomd. Verder vindt u op deze pagina uitleg en screenshots over de applicatie, zodat u een beeld krijgt hoe het eruit ziet en wat het doet.

De vacature applicatie is totaal dynamisch en past zich aan als een gebruiker de data verandert, zoals het aantal open vacatures, gesloten vacatures etc. Er is gebruik gemaakt van responsive design, zodat de gebruikers op alle devices de applicatie makkelijk kunnen bekijken. Daarnaast is er rekening gehouden met de UX voor een duidelijk overzicht. U kunt deze applicatie downloaden om als web applicatie te gebruiken, zie de afbeeldingen 5.1 en 5.2 onderaan deze pagina.

Technische gegevens:

  1. Back-end : Php (deels OOP), mysql. Php-versie: 7.4.28
  2. Front-end: Html5, Css3, Bootstrap, Javascript, Custom Components.
  3. Framework: Angularjs 1.6.2.
  4. Data: Sql queries en Json API.
  5. Progressive Web App (PWA) : Installeer en gebruik de website als een (web) app.

Uitleg over de applicatie:

Afbeelding 1 is een screenshot van de home pagina van mijn vacature applicatie. Op de home pagina van de applicatie kan een gebruiker een vacature toevoegen, verwijderen, wijzigen of details van een vacature lezen.

Afbeelding 2 hieronder toont de vacature-overzicht pagina en alle toegevoegde vacatures die een gebruiker heeft toegevoegd. Deze pagina bevat filteropties die een gebruiker kan filteren op categorie, niveau en status. Verder kan een gebruiker een vacature zoeken op vacature-nummer, title, categorie, plaats en andere beschikbare gegevens. De onderstaande pagina is een html pagina en is gebouwd met Angular 1.6.2. De gegevens worden uit de mysql database opgehaald en omgezet in Json format. Vervolgens worden de gegevens met ng-repeat op het scherm afgedrukt.

Afbeelding 3 laat de dashboard-pagina zien. De onderstaande grafiek is dynamisch en past zich automatisch aan als de data verandert. Voor deze grafiek is gebruik gemaakt van Google charts.

Afbeelding 4 toont een login pagina. Om mijn CV te zien dient een gebruiker in te loggen met een vooraf gegenereerde code. Deze code heeft een duur van 7 dagen. Na 7 dagen vervalt de code. Om de code te heractiveren, kan ik de datum via de database verlengen.
