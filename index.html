Opdrachbeschrijving:

Casus

Een applicatie om crypto currency te administreren. Een gebruiker kan verschillende soorten crypto currency toevoegen aan zijn persoonlijke overzicht en heeft een overzicht van de hoeveelheid crypto van de verschillende soorten in bezit zijn en de totale waarde ervan. Resources

Om een overzicht op te halen van de bekende crypto coins is de Coingcko API gebruikt. Deze service levert een lijst van alle crypto munten.

Database:

De collectie van munten van de gebruiker wordt in de local storage van de webbrowser opgeslagen. De benadering van de local storage verloopt via een (asynchrone) service laag.

Gerealiseerde applicatie onderdelen:

De meeste gerealiseerde applicatie onderdelen zijn door middel van (custom) web componenten gebouwd.

Hieronder volgen de gerealiseerde onderdelen:

Elke pagina is uitsluitend beschikbaar via een route. De applicatie is volledig SPA (Single Page Application)

  • Header
  • Grafiek (pie-chart)
  • Muntenoverzicht : lijst met crypto munten die in het bezit zijn van de gebruiker.
  • Details button.
  • Een door de gebruiker in te vullen crypto hoeveelheid
  • Buttons om toe te voegen of te annuleren.
